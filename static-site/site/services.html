Capabilities

Cross-platform engineering for difficult systems

I work across identity, servers, virtualization, storage, networking, cloud services, security controls, backup platforms, and emerging AI infrastructure.

The emphasis is not on selling a particular product. It is on understanding the environment, reducing operational risk, and choosing a practical path forward.

Infrastructure modernization

Modernize without losing the recovery path

Assess and modernize aging or unsupported systems while protecting service continuity and operational knowledge.

  • Current-state discovery and dependency mapping
  • Windows Server and domain-controller modernization
  • File, print, application, and ERP dependency analysis
  • Staged cutover, rollback, and validation plans

Virtualization and migration

Move workloads with storage and service dependencies in view

Plan physical-to-virtual, virtual-to-virtual, and platform migrations around source protection, storage ownership, identity, and application behavior.

  • VMware ESXi and vCenter
  • Microsoft Hyper-V
  • Proxmox VE and Proxmox Backup Server
  • Virtual-disk conversion, recovery, and validation

Microsoft cloud and identity

Troubleshoot the full identity and application path

Work across Microsoft 365, Exchange Online, Entra ID, Azure, Azure Virtual Desktop, Windows hosts, profiles, tokens, and application integrations.

  • Conditional Access and authentication
  • Mail flow, connectors, OAuth, and app registrations
  • AVD host, domain, profile, audio, and application issues
  • OneDrive, SharePoint, Outlook, and Teams integrations

Network and security engineering

Trace and control the path end to end

Design and troubleshoot routing, segmentation, remote access, published services, and monitoring without treating the firewall as an isolated appliance.

  • Sophos, Juniper, and Meraki environments
  • VLANs, routing, NAT, DNAT, SNAT, and DNS
  • Secure remote access and Tailscale
  • Security Onion, Zeek, Elastic, IDS, and SIEM workflows

Storage, backup and continuity

Make recovery evidence part of the plan

Protect data and recovery options during migrations, failures, and platform changes.

  • Dell PowerVault and enterprise SAN environments
  • SAS, iSCSI, multipath I/O, LUN mapping, and presentation
  • ReFS, NTFS, ZFS, VMFS, and virtual disks
  • Veeam, restore validation, and continuity runbooks
